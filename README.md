# OAuth 2.0 Client Credentials Demo (Genesys Cloud)

Simple Node.js app demonstrating how to:

- Authenticate using OAuth 2.0 Client Credentials
- Cache access tokens in memory
- Call Genesys Cloud APIs
- Organize code into small reusable services

Built with:
- Node.js
- Express
- Axios
- dotenv
